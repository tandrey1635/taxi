# Лендинг «CabHub» — сайт заказа такси
Верстка лендинга по PSD-макету для сервиса заказа такси на чистом HTML5/SASS с адаптивной версткой.


## 🛠 Технологический стек

### Frontend
- **HTML5** — семантическая верстка, БЭМ-методология
- **SASS/SCSS** — сброс стилей, медиа-запросы


## 🎯 Ключевые функции

### UI/UX элементы
- Адаптивное бургер-меню на мобильных устройствах через тег details
- Форма бронирования такси с выбором времени, даты, точки отправления и назначения
- Карточки тарифов с выделением активного
- Секция преимуществ с иконками
- Секция отзывов клиентов
- Секция загрузки мобильного приложения
- Контактная информация и социальные сети


## 🔧 Технические особенности

### Адаптивность
- Адаптивный дизайн
- Медиа-запросы
- Кросс-браузерная совместимость


**Проект демонстрирует навыки верстки по PSD-макету на чистом HTML5/SASS с адаптивным дизайном.**
